# Features

## Aimbot

- Aimbot FOV
- Aimbot Smoothness
- Aimbot Prediction
- Aimbot Bullet Drop (WIP)

## ESP

- Name ESP
- Chams ESP
- Tracer
- Text Visible Check
- Chams Visible Check
- Tracer Visible Check
- Chams Color
- Text Color
- Tracer Color

"X Visible Check" means that it will only show if the player is visible.

- Hitbox Size
- Togglee Part (Head or Torso) With A Key
